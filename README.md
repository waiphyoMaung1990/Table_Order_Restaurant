# Table_Order_Restaurant

This repository is for the **Table Order Restaurant System** project.

## Overview

The Table Order Restaurant System is designed to help restaurants streamline the process of customers ordering food and drinks from their tables using a mobile app or web interface. It supports cart management, quantity selection, Kyat-based payment, and table management functions.

## Features

- Add food and drinks to a cart
- Specify quantity for each item
- Pay directly from the table using Kyats
- Table management for restaurant staff
- Easy-to-use user and admin interfaces
